# docs/plans/

Historical session artifacts generated by the `superpowers:writing-plans` skill during early release cycles (pre-v0.30.0 era, roughly February through April 2026). Each file is a release-cycle implementation plan corresponding to a specific feature or modality addition.

## NOT current architecture

These files describe implementation steps as originally planned. They are historical records, not current truth. For current architecture see:

1. `CLAUDE.md` (repo root) -- the authoritative project narrative, kept current
2. `docs/CHAT_COMPLETION.md`, `docs/HF_PLUGINS.md`, `docs/MODEL_SCRIPTS.md`, `docs/RESOLVERS.md` -- current architecture references

## Relationship to docs/superpowers/

Later release cycles (v0.30.0+) moved plan and spec files to `docs/superpowers/plans/` and `docs/superpowers/specs/`. This directory covers the earlier period before that convention was established. The content here is analogous in nature: session ephemera, not living documentation.
