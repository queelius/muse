# muse example catalog

Outputs produced by exercising a running `muse serve` instance across modalities.
Each modality has its own directory containing the generated artifact(s) plus a
`metadata.json` recording full provenance (model, endpoint, request params,
timing, muse version, server commit, timestamp).

Generated by exercising muse v0.47.0 (commit recorded per-item in metadata.json).

## Layout

    examples/
      <modality>/
        <artifact files>
        metadata.json      # provenance for this modality's run(s)
      _logs/               # server + driver logs
      INDEX.md             # summary table of what was exercised

See INDEX.md for the run summary (status per modality, including any errors hit
and how they were resolved).
